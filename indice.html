--- layout: default ref: index lang: es --- Aprende, Colabora y Dockeriza! En tu comunidad muchos desarrolladores y ingenieros de sistemas estan usando y aprendiendo sobre Docker, ven a conocerlos. Docker es una plataforma abierta que ayuda a crear, enviar y ejecutar aplicaciones en cualquier momento y lugar. Los desarrolladores usan Docker para modificar el código y agilizar el desarrollo de aplicaciones, mientras que los ingenieros de operaciones obtienen soporte para responder rápida y flexiblemente a necesidades cambiantes. Docker garantiza agilidad, portabilidad y control para todas sus aplicaciones distribuidas. Docker es el principal patrocinador del Proyecto Moby, un marco abierto para ensamblar sistemas de contenedores, sin tener que reinventar la rueda.
